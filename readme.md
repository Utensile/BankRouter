# BankRouter (Turtle WoW)

A custom addon for Turtle WoW (Vanilla 1.12.1) that adds a button to the mailbox interface. 

## Features
- Scans bags for specific items (e.g., Runecloth, Thorium).
- Automatically queues and sends them to configured bank alts.
- Throttle protection to prevent server disconnects.

## Installation & Configuration
1. Copy the `BankRouter` folder into `WoW/Interface/AddOns/`.
2. Edit `BankRouter.lua` to configure your item lists and character names.
